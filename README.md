home-agency-style-reen
======================

## For beginning ##
```
npm install
grunt build
```

## For developing ##
```
grunt
```

## Used ##
- Sass (SCSS)
- Bootstrap 3
- Haml
- Grunt, include this plugins:
  - grunt-contrib-haml
  - grunt-contrib-sass
  - grunt-contrib-watch
  - grunt-yui-compressor
